# 🎙️ Snippet

**Say it once. Post it everywhere.**

Snippet is an AI-powered marketing assistant that transforms a single voice recording into platform-specific marketing content. Instead of manually writing posts for multiple channels, business owners simply record one voice memo, and Snippet generates ready-to-use content for Instagram, WhatsApp, and SMS.

🌐 **Live Demo:** https://snippet-steel.vercel.app/

## 🚀 How It Works

1. Select a business scenario.
2. Click **Press Record & Generate Kit**.
3. Snippet simulates voice processing.
4. AI generates:
   - Instagram Reel Script
   - WhatsApp Broadcast
   - SMS Message
5. Copy, edit, and publish.
